# Frontend - Student AI Assistant

React + Vite + Tailwind CSS frontend for the Student AI Assistant application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Features

- React Router for navigation
- Tailwind CSS for styling
- Axios for API calls
- JWT token management
- Responsive design

## Pages

- `/login` - Login page
- `/signup` - Signup page
- `/dashboard` - Main dashboard (protected)

## Components

- `AIChatbox` - Full-screen AI chat interface
- `ComplaintModal` - Modal for raising complaints
- `Dashboard` - Main dashboard with all features
